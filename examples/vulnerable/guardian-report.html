CVE-2021-23337

lodash

v4.17.15 → v4.18.1 expand_more

Clique no card para ver os detalhes.

Por que isso importa

`lodash` versions prior to 4.17.21 are vulnerable to Command Injection via the template function.

arrow_right

Como acontece

Não há evidência forte de reachability, mas a dependência continua afetada e existe exposição potencial por superfície externa segundo a fonte osv e os advisories associados.

arrow_right

Impacto

O impacto potencial está relacionado a "Command Injection in lodash", com severidade high.

arrow_right

Ação

Atualize lodash da versão 4.17.15 para 4.18.1 e valide os fluxos principais após a mudança.
